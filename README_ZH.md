**Sidefy**是一款 macOS 菜单栏应用程序，将来自各种来源（日历、提醒事项、RSS、GitHub 等）的事件和信息统一到一个动态屏幕边缘信息流中。

>[!WARNING] 
>不再提供 Github Release，最新版请在 AppStore 下载。
